# dotfiles

> work in progress

My dotfiles.

## Installation

Install them using [`stow`](https://www.gnu.org/software/stow/).

```sh
# Clone repository to ~/.dotfiles
git clone https://github.com/lukadev-0/dotfiles ~/.dotfiles

# Switch to ~/.dotfiles
cd ~/.dotfiles

# Run stow to install dotfiles
stow ags dunst hyprland neovim zsh
```
